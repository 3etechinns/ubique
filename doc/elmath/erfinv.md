### erfinv

Inverse error function


#### Syntax

ubique.erfinv(y)


#### Description

Inverse error function  
  
The inverse error function satisfies y = erf(x), for -1 <= y <= 1 and -inf <= x <= inf  



#### Arguments

|Params|Type|Description
|---------|----|-----------
|`y` | number | real value in range [-1,1]


#### Examples

```js
ubique.erfinv(0.1);
// 0.08885596505119535
```
