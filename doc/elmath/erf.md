### erf(x)

Error function


#### Description

Error function  
  
Approximation with maximal error 1.2x10^-7  
erf(x) = 2/sqrt(pi) * integral from 0 to x of exp(-t^2) dt  



#### Arguments

|Params|Type|Description
|---------|----|-----------
|`x` | number | real value


#### Examples

```js
ubique.erf(0.5);  // 0.5204999077232426
```
