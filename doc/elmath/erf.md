### erf
Error function


#### Description

Error functionApproximation with maximal error 1.2x10^-7erf(x) = 2/sqrt(pi) * integral from 0 to x of exp(-t^2) dt


|Params|Type|Description
|---------|----|-----------
|`x` | number | real value


#### Examples

```js
ubique.erf(0.5);  // 0.5204999077232426
```
