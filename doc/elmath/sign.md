### sign(x)

Signum function


#### Description

Signum function  



#### Arguments

|Params|Type|Description
|---------|----|-----------
|`x` | number/array/matrix | element


#### Examples

```js
ubique.sign(-0.5);  // -1
ubique.sign([0.1,-0.5]); // [1, -1]
ubique.sign([[5,-2],[-3,4]]); // [[1, -1], [-1, 1]]
```
