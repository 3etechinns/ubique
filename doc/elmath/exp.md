### exp(x)

Exponential value


#### Description

Exponential value  



#### Arguments

|Params|Type|Description
|---------|----|-----------
|`x` | number/array/matrix | element


#### Examples

```js
var a = [[5,6,5],[7,8,-1]];
var c = [5,6,3];

ubique.exp(6); // 403.429
ubique.exp(c); // [148.413, 403.429, 20.0855]
ubqie.exp(a); // [[148.413, 403.429, 148.413], [1096.63, 2980.96, 0.367879]]
```
