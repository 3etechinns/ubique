### exp

Exponential value


#### Syntax

ubique.exp(x)


#### Description

Exponential value  



#### Arguments

|Params|Type|Description
|---------|----|-----------
|`x` | number/array/matrix | element


#### Examples

```js
ubique.exp(6);
// 403.429

ubique.exp([5,6,3]);
// [148.413, 403.429, 20.0855]

ubqie.exp([[5,6,5],[7,8,-1]]);
// [[148.413, 403.429, 148.413], [1096.63, 2980.96, 0.367879]]
```
