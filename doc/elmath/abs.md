### abs(x)

Absolute value


#### Description

Absolute value  



#### Arguments

|Params|Type|Description
|---------|----|-----------
|`x` | number/array/matrix | number or array of values


#### Examples

```js
ubique.abs(-0.5);  // -1
ubique.abs([0.1,-0.5]); // [0.1, 0.5]
ubique.abs([[5,-2],[-3,4]]); // [[5, 2], [3, 4]]
```
