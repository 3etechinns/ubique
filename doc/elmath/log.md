### log

Square root


#### Syntax

ubique.log(x)


#### Description

Square root  



#### Arguments

|Params|Type|Description
|---------|----|-----------
|`x` | number/array/matrix | element


#### Examples

```js
ubique.sqrt(6);
// 2.44949
ubique.sqrt([5,6,3]);
// [ 2.23607, 2.44949, 1.73205 ]
ubique.sqrt([[5,6,5],[7,8,2]]);
// [ [ 2.2361, 2.4495, 2.2361 ], [ 2.6458, 2.8284, 1.4142 ] ]
```
