### det

Matrix determinant


#### Syntax

ubique.det(x)


#### Description

Matrix determinant with LU decomposition method  



#### Arguments

|Params|Type|Description
|---------|----|-----------
|`x` | matrix | square matrix


#### Examples

```js
ubique.det([[1,5],[6,2]]);
// -28
ubique.det([[2,2],[2,3]]);
// 2
```
