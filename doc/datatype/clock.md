### clock

Current date and time as date vector


#### Syntax

ubique.clock()


#### Description

Current date and time as date vector. Returns an array:  
[year month day hour minute seconds]  



#### Arguments

|Params|Type|Description
|---------|----|-----------
|`empty` | empty | no arguments


#### Examples

```js
ubique.clock();
// [ 2015, 5, 28, 11, 51, 0, 801 ]
```
