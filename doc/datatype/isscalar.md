### isscalar

True for scalar input


#### Syntax

ubique.isscalar(x)


#### Description

True for scalar input  



#### Arguments

|Params|Type|Description
|---------|----|-----------
|`x` | number/array/matrix | input


#### Examples

```js
ubique.isscalar(2);
// true

ubique.isscalar([2]);
// true

ubique.isscalar([[2]]);
// true
```
