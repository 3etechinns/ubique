### ismatrix

True for matrix


#### Syntax

ubique.ismatrix(x)


#### Description

True for array of array (matrix)  



#### Arguments

|Params|Type|Description
|---------|----|-----------
|`x` | matrix | matrix


#### Examples

```js
ubique.ismatrix([[1,3,4]]);
// true

ubique.ismatrix([[1],[3],[4]]);
// true
```
