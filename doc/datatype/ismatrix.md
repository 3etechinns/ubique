### ismatrix(x)

True for matrix


#### Description

True for array of array (matrix)  



#### Arguments

|Params|Type|Description
|---------|----|-----------
|`` | matrix | x matrix


#### Examples

```js
ubique.ismatrix([[1,3,4]]); // true (1x3)
ubique.ismatrix([[1],[3],[4]]); // true (3x1)
```
