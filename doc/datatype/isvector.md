### isvector

True for vector input


#### Syntax

ubique.isvector(x)


#### Description

True for vector input  



#### Arguments

|Params|Type|Description
|---------|----|-----------
|`x` | matrix | matrix Nx1 or 1xN


#### Examples

```js
ubique.isvector([[5,6,7]]);
// true
ubique.isvector([[5],[6],[7]]);
// true
```
