### isnan

True only for NaN input


#### Syntax

ubique.isnan(x)


#### Description

True only for NaN input. For [null,true,'hello',5,{},undefined,'NaN'] returns False.  



#### Arguments

|Params|Type|Description
|---------|----|-----------
|`x` | NaN | element


#### Examples

```js
ubique.isnan(NaN);
// true
```
