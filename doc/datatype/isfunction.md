### isfunction

True for function


#### Syntax

ubique.isfunction = 


#### Description

true for function  



#### Arguments

|Params|Type|Description
|---------|----|-----------
|`x` | function | function


#### Examples

```js
ubique.isfunction(function(a){return console.log(a);});
// true
ubique.isfunction(Math.log);
// true
```
