### clone

Create a clone of the input array or matrix


#### Syntax

ubique.clone(x)


#### Description

Create a clone of the input array or matrix  



#### Arguments

|Params|Type|Description
|---------|----|-----------
|`x` | array/matrix | array or matrix


#### Examples

```js
ubique.clone([[-1,3,-1],[4,5,9]]);
// [ [ -1, 3, -1 ], [ 4, 5, 9 ] ]

ubique.clone([5,6,3]);
// [ 5, 6, 3 ]
```
