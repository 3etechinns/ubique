### ndims

Number of array dimensions


#### Syntax

ubique.ndims(x)


#### Description

Number of array dimensions  



#### Arguments

|Params|Type|Description
|---------|----|-----------
|`x` | array/matrix | array of elements


#### Examples

```js
ubique.ndims([3,5,6]);
// 2

ubique.ndims([[3,2,7],[4,5,6]]);
// 2
```
