### row

Get a row of matrix


#### Syntax

ubique.row(x,n)


#### Description

Get a row of a matrix  



#### Arguments

|Params|Type|Description
|---------|----|-----------
|`x` | array/matrix | matrix of elements
|`n` | number | row number (indexing from 0 to n - 1)


#### Examples

```js
ubique.row([[5,6,5],[7,8,-1]],0);
// [5, 6, 5]
ubique.row([5,6,5]);
// 5
```
