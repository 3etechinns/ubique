### ncols

Number of columns in vector|matrix MxN


#### Syntax

ubique.ncols(x)


#### Description

Number of columns in vector|matrix. Array ~ Row vector, returns N.  



#### Arguments

|Params|Type|Description
|---------|----|-----------
|`x` | array/matrix | array of elements


#### Examples

```js
ubique.ncols([5,6,7]);
// 3

ubique.ncols([[3,2,7],[4,5,6]]);
//  3
```
