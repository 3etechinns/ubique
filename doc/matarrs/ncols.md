### ncols(x)

Number of columns in vector|matrix


#### Description

Number of columns in vector|matrix  



#### Arguments

|Params|Type|Description
|---------|----|-----------
|`x` | array/matrix | array of elements


#### Examples

```js
ubique.ncols([5,6,7]); // 1
ubique.ncols([[3,2,7],[4,5,6]]); //  3
```
