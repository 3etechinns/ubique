### linspace

Create linearly spaced arrays


#### Syntax

ubique.linspace(a,b,n)


#### Description

Create linearly spaced arrays  



#### Arguments

|Params|Type|Description
|---------|----|-----------
|`a` | number | lower bound
|`b` | number | upper bound
|`n` | number | number of points


#### Examples

```js
ubique.linspace(1,10,5);
// [1, 3.25, 5.5, 7.75, 10]
```
