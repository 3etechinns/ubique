### length
length of vector or largest array dimension


#### Description

length of vector or largest array dimension


|Params|Type|Description
|---------|----|-----------
|`x` | array/matrix | array of elements


#### Examples

```js
ubique.length([3,5,6]); // 3
ubique.length(5); // 1
ubique.length([[5,4],[-1,2]]); // 2
```
