### nrows

Number of rows in vector|matrix MxN


#### Syntax

ubique.nrows(x)


#### Description

Number of rows in vector|matrix. Array ~ Row vector, returns 1.  



#### Arguments

|Params|Type|Description
|---------|----|-----------
|`x` | array/matrix | array of elements


#### Examples

```js
ubique.nrows([5,6,7]);
// 1

ubique.nrows([[3,2,7],[4,5,6]]);
// 2
```
