### rows

Number of rows in vector|matrix


#### Syntax

ubique.rows(x)


#### Description

Number of rows in vector|matrix  



#### Arguments

|Params|Type|Description
|---------|----|-----------
|`x` | array/matrix | array of elements


#### Examples

```js
ubique.nrows([5,6,7]);
// 3
ubique.nrows([[3,2,7],[4,5,6]]);
// 2
```
