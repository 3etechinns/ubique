### numel(x)

Number of elements in an array


#### Description

Number of elements in an array  



#### Arguments

|Params|Type|Description
|---------|----|-----------
|`x` | array/matrix | array of elements


#### Examples

```js
ubique.numel([3,5,6]); // 3
ubique.numel([[3,2,7],[4,5,6]]); // 6
```
