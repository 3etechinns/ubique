### numel

Number of elements in an array


#### Syntax

ubique.numel(x)


#### Description

Number of elements in an array  



#### Arguments

|Params|Type|Description
|---------|----|-----------
|`x` | array/matrix | array of elements


#### Examples

```js
ubique.numel([3,5,6]);
// 3
ubique.numel([[3,2,7],[4,5,6]]);
// 6
```
