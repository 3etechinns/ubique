### col
Get a column of a matrix


#### Description

Get a column of a matrix


|Params|Type|Description
|---------|----|-----------
|`x` | array/matrix | matrix of elements
|`n` | number | column number (indexing from 0 to n - 1)


#### Examples

```js
ubique.col([[5,6,5],[7,8,-1]],0); // [5, 7]
```
