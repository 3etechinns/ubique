### tomat
Convert number or array to matrix


#### Description

Convert number or array to matrix


|Params|Type|Description
|---------|----|-----------
|`x` | number/array/boolean | input


#### Examples

```js
ubique.tomat(5); // [[5]]
ubique.tomat([5,6,3]); // [[5], [6], [3]]
ubique.tomat(true); // [[true]]
```
