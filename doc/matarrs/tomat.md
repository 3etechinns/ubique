### tomat

Convert number or array to matrix


#### Syntax

ubique.tomat(x)


#### Description

Convert number or array to matrix  



#### Arguments

|Params|Type|Description
|---------|----|-----------
|`x` | number/array/boolean | input


#### Examples

```js
ubique.tomat(5);
// [[5]]
ubique.tomat([5,6,3]);
// [[5], [6], [3]]
ubique.tomat(true);
// [[true]]
```
