### uminus
Unary minus -X


#### Description

Unary minus -X


|Params|Type|Description
|---------|----|-----------
|`x` | number/array/matrix | number or array of values


#### Examples

```js
ubique.uminus(-5); // 5
ubique.uminus([5,6]); // [-5, -6]
ubique.uminus([[5,6],[-1,-3]]); // [[-5, -6], [1, 3]]
```
