### uplus(x)

Unary plus +X


#### Description

Unary plus +X  



#### Arguments

|Params|Type|Description
|---------|----|-----------
|`x` | number/array/matrix | number or array of values


#### Examples

```js
ubique.uplus(-5); // -5
ubique.uplus([5,6]); // [5, 6]
ubique.uplus([[5,6],[-1,-3]]); // [[5, 6], [-1, -3]]
```
