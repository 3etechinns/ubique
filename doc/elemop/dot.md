### dot

Arrays dot product X * Y


#### Syntax

ubique.dot(x,y)


#### Description

Arrays dot product X * Y. X and Y must be arrays of the same length  



#### Arguments

|Params|Type|Description
|---------|----|-----------
|`x` | array | number or array of values
|`y` | array | number or array of values


#### Examples

```js
var c = [5,6,3];
var d = [0.5,-3,2.3];

ubique.dot(c,d);
// -8.6
```
