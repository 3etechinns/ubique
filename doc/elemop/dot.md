### dot
Arrays dot product X * Y


#### Description

Arrays dot product X * Y. X and Y must be arrays of the same length


|Params|Type|Description
|---------|----|-----------
|`x` | `number|array` | number or array of values
|`y` | `number|array` | number or array of values


#### Examples

```js
var c = [5,6,3];
var d = [0.5,-3,2.3];

ubique.dot(5,6); // 30
ubique.dot(c,d); // -8.6
```
